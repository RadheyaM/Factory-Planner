How To Use The Packing Calculator

Navigation

Clicking Packing Calculator takes you to the information for the current active week provided you are logged in

The information on this page can be accessed for any plan by clicking view on the desired plan in the 'plan search page'

This page contains all relevant reports for the week plan

Other links on the left take you to their respective search pages

Plan Creation Instructions

Plans are made up of runs, runs are made up of products and products need a packing configuration

  1. Create a Packing Configuration
  2. Create a Product, assigning the relevant packing configuration
  3. Create a Run, assigning a product to the run
  4. Create a Week Plan, Click Add Run
  5. Assign as many pre-created runs as required to the Plan
